# AudacityTees — 50 Design Generation Prompts (v2 — Exact Instructions)

## MASTER RULES FOR EVERY PROMPT
Prepend this to every prompt if your AI doesn't carry context between generations:
```
TECHNICAL REQUIREMENTS:
- Output: PNG image, 4500x5400 pixels, 300 DPI
- Background: 100% transparent (alpha channel). No white background. No colored background. TRANSPARENT ONLY.
- The design will be printed on a t-shirt using DTG printing. The background of the shirt provides the "canvas" — the PNG must only contain the design elements with transparency everywhere else.
- Center the design horizontally. Place it in the upper-center area of the canvas (typical chest placement for a t-shirt print area).
- Leave at least 400px margin on all sides — do not let any element touch the edges of the canvas.
```
