# ShirtyNation Design Research

This folder contains market research data used to drive design decisions.
Updated continuously as new trends emerge.

## Files
- `shirt-concepts.md` — Master list of validated shirt ideas with data sources
- `niche-analysis.md` — Category-level insights (oversaturated vs opportunity)
- `pricing-strategy.md` — Price point data across platforms
- `design-queue.md` — Prioritized queue for GPT-5.4 generation (batch 1, 2, 3...)
